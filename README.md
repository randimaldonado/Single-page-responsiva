# Single-page-responsiva-empresa-generica
Esta é uma single page responsiva que criei a partir do zero, com pensamento mobile firts, usando somente HTML e CSS, sem nenhum tipo de framework, ela utiliza as propriedades flex e @media para ser totalmente responsiva.
